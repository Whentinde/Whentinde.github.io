Hexo搭建一个自己的博客

“其他”
Publish Date:   2022-06-06

第一篇文章搭建博客（老传统）。

Node.js下载

前往nodejs官网下载node.js（左边）

win+R cmd

node -v检查是否下载完成

npm -v同上检查

npm install -g cnpm --registry=http://registry.npm.taobao.org下载淘宝镜像源

下载完成

Hexo下载

全局安装Hexo

Git下载

前往git官网

点这个，下载git

退回桌面唤出终端

git --version检查是否安装好

会是这个样子

Github搭建仓库

注册一个Github

通过Github的仓库作为服务器，把博客内容寄存至Github仓库，实现博客搭建。

最重要的是这样的服务器不要钱。。。

Github首页new一个

写入自己的注册名如：“xxx.github.io”

勾选添加阅读文件

Create

仓库搭建完成

开始搭建工作

打开git终端

首先生成一个ssh Keys

ssh-keygen -t rsa -C “邮箱”

找到用户中的.ssh文件

id_rsa.pub复制里面的内容

Setting

SSH and GPG Keys

粘贴至公钥里，title自行起名

git 绑定密钥

mkdir blog创建blog文件夹

cd blog/移步blog文件夹

pwd查看当前路径是否已至blog

hexo init初始化你的博客（这步骤可能会错误，可能是连接超时，多尝试几次）

发生其他问题的话移步百度

这样就成功了

hexo s静态生成博客

通过此连接可以与自己的第一个博客见面了

它长这样。

还未完成，

进入blog文件夹修改_config.yml

改这个

repository填写自己GitHub code的位置

cnpm install hexo-deployer-git --save安装自动部署工具

hexo g生成页面

hexo d上传

这是已经传过的

输入用户名

password需要github博客令牌

通过开发者设置生成一个令牌

输入

完成

接下来就可以通过注册的网址访问自己的博客了
